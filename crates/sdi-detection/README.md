# sdi-detection

Native Leiden community detection for [sdi-rust](https://github.com/geoffgodwin/sdi-rust).

Pure Rust port of the Leiden algorithm (Traag et al. 2019, Modularity + CPM quality function).
No FFI, no C++ dependency — deterministic across platforms with a seeded `StdRng`.

Part of the `sdi-rust` workspace. See the [workspace README](../../README.md) for full documentation.
