# sdi-lang-javascript

JavaScript language adapter for [sdi-rust](https://github.com/geoffgodwin/sdi-rust).

Implements `LanguageAdapter` using the `tree-sitter-javascript` grammar to parse JavaScript
source files and produce `FeatureRecord`s for the sdi-rust pipeline.

Part of the `sdi-rust` workspace. See the [workspace README](../../README.md) for full documentation.
