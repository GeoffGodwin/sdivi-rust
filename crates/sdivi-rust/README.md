# sdivi-rust

Install-discovery meta-crate for the [Structural Divergence Indexer](https://github.com/geoffgodwin/sdivi-rust).

```sh
cargo install sdivi-rust
sdivi --help
```

This meta-crate re-exports `sdivi-core` and serves as the `cargo install` entry point.
The `sdivi` binary is defined in `sdivi-cli`.

See the [workspace README](../../README.md) for full documentation.
