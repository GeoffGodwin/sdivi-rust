# sdi-rust

Install-discovery meta-crate for the [Structural Divergence Indexer](https://github.com/geoffgodwin/sdi-rust).

```sh
cargo install sdi-rust
sdi --help
```

This meta-crate re-exports `sdi-core` and serves as the `cargo install` entry point.
The `sdi` binary is defined in `sdi-cli`.

See the [workspace README](../../README.md) for full documentation.
