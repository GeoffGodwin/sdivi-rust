# sdi-lang-rust

Rust language adapter for [sdi-rust](https://github.com/geoffgodwin/sdi-rust).

Implements `LanguageAdapter` using the `tree-sitter-rust` grammar to parse Rust source
files and produce `FeatureRecord`s for the sdi-rust pipeline.

Part of the `sdi-rust` workspace. See the [workspace README](../../README.md) for full documentation.
