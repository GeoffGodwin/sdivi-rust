# sdi-lang-typescript

TypeScript language adapter for [sdi-rust](https://github.com/geoffgodwin/sdi-rust).

Implements `LanguageAdapter` using the `tree-sitter-typescript` grammar to parse TypeScript
source files and produce `FeatureRecord`s for the sdi-rust pipeline.

Part of the `sdi-rust` workspace. See the [workspace README](../../README.md) for full documentation.
